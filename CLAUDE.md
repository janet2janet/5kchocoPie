# 5kChocoPie

## What this is
A Y2K-themed short film production planning website that helps filmmakers manage budgets (camera, gear, actors, crew) and organize their film projects in a fun, easy-to-use interface.

The site has a Windows XP / early-web Y2K aesthetic: pixel fonts (Press Start 2P, VT323), neon-on-dark color palette, WinXP-style window chrome with gradient title bars, raised bevel buttons, and blinking cursor decorations.

Data is persisted via a local `projects.json` file served by a tiny Express backend on port 3001. The Vite frontend (port 5173) proxies `/api` calls to it. Run both with `npm start`.
